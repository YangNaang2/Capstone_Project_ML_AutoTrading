# 🏆 2025년도 소프트웨어융합대학 캡스톤 FAIR **최우수상 (2위)** 수상

## 머신러닝을 이용한 암호화폐 변동성 예측과 Automated Trading System (UPbit)

**2024 Fall & 2025 Spring**

## 📌 Project Overview

   가치 투자와 기술적 분석 중 기술적 분석으로 접근<br>
   LSTM (Long Short-Term Memory)과 GRU (Gated Recurrent Units)와 같은 머신러닝 모델을 사용하여 가격 변동을 예측<br>
   여러 트레이딩 전략들을 프로그래밍하고, **Python 라이브러리인 Backtrader**를 이용해 백테스팅을 진행<br> 
   백테스팅 결과를 토대로 실제 시장에서 여러 전략들이 어떻게 통용되는지 알아보고자 함<br>

최종적으로는 **UPbit API**를 활용해 실시간 거래 시스템을 구축하고, 여러 전력들을 조합, 분해하여 우리 만의 전략을 구현하는 것이 최종 목표

---

## ⚙️ 주요 기능

1. **머신러닝 모델**  
   - **LSTM**과 **GRU** 모델을 활용하여 암호화폐 가격의 변동을 예측.
   - 모델 성능 평가 및 하이퍼파라미터 조정을 통한 최적화.
   
2. **백테스팅**  
   - **Backtrader**를 사용해 여러 거래 전략들을 백테스트하여 전략의 효율성을 평가.
   
3. **자동화 거래**  
   - **UPbit API**와 연동하여 실시간 시장 데이터로 거래 실행.

---

## 🚀 프로젝트 흐름

- **2024년 8월**: **투자 방식 공부**  
  - **가치 투자**와 **기술적 분석** 에 대한 이해. 투자 방식 및 프로젝트 진행 방향 설정.
    
- **2024년 9월**: **GRU, LSTM 구현화**  
  - **GRU**와 **LSTM** 모델을 구현하여 암호화폐 변동성(가격) 예측을 위한 모델을 학습하고 테스트.

- **2024년 10월**: **전략 공부**
  - 전통적인 거래 전략들을 공부하여 투자에 대한 기초 지식을 배움.

- **2024년 11월 ~ 12월**: **Backtrader 공부** 
  - **Backtesting** 개념 습득.
  - **yfinance** 통해 과거 주가 데이터 불러오고 **Backtrader** 를 활용하여 백테스팅 진행.

- **2025년 1월**: **사용할 전략 선택**  
  - Backtesting 결과 기반으로 최종 사용 전략을 선택. 해당 전략들의 최적화를 진행.

- **2025년 2월 ~ 3월**: **여러 전략 구현 및 웹사이트 개설**  
  - 선택된 전략들의 Backtesting 결과와 Trading View의 api를 결합하여 웹 사이트 개설.

- **2025년 4월**: **UPbit API와 전략 결합**  
  - UPbit의 OPEN API 키를 발급받아 선택된 전략들과 결합, 실제 시장에서 전략들 실행.

- **2025년 5월**: **전략 개발**  
  - 전략들의 결합과 분해, 파라미터 변경을 통한 신규 전략 개발.
  - 일반 트레이더들의 전략들 참고하여 코드 구현화.

- **2025년 6월**: **캡스톤 프로젝트 종료**  
  - 6월 4일 캡스톤 프로젝트 최종 발표
